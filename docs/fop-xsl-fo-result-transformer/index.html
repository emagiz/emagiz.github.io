fop-xsl-fo-result-transformer

FOP XSL-FO result transformer

Converts the result of an XSL-FO transformation into a PDF document (represented as a byte array).

Converts the resulting PDF document of a FOP XSL-FO transformation into a byte array (other Result implementations are not supported).

Can be used in combination with FOP XSL-FO result factory to generate PDF documents using Spring Integration's XSLT transformer.
