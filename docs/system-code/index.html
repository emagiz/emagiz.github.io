System code

System

The system this code is specified for.

Required

Code type

The code type of this system code.

Required

CDM code

The CDM code this system code is mapped to.

Required

System code

Unique value for this system code.

This value needs to match exactly the value used in the calls to the mapping web services that are made from eMagiz message flows.

Required

Primary (CDM > system)

Whether this system code is the primary (CDM > system) code or not.

Because multiple system codes can be mapped to multiple CDM codes (many-to-many), one of them must be flagged as primary (CDM > system). When a message flow needs to map a CDM code to a system code, this primary (CDM > system) code will be selected.

Primary (system > CDM)

Whether this system code is the primary (system > CDM) code or not.

Because multiple system codes can be mapped to multiple CDM codes (many-to-many), one of them must be flagged as primary (system > CDM). When a message flow needs to map a system code to a CDM code, this primary (system > CDM) code will be selected.
