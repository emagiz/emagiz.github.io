xmpp-inbound-channel-adapter

id: title: XMPP inbound channel adapterXMPP inbound channel adapter

Receives chat messages sent to a given account.

Documentation

Configures an endpoint that will receive chat messages sent to a given account and then forward those messages to a MessageChannel.

Payload expression

A SpEL expression to evaluate a payload with the incoming org.jivesoftware.smack.packet.Message as root object. Useful in case of custom (XEP) XMPP interactions, e.g. GCM.

By default a message body is used as payload. The #extension SpEL variable is registered in the evaluation context if one and only one extension is present in the message.

Error channel

The channel that error messages will be sent to if a failure occurs in this adapter's invocation.
