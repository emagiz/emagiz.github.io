header-filter

Header filter

Header names

Specify one or more header names (as a comma separated list) to be removed from the messages handled by this transformer.

Required

Pattern match

Whether values provided in header names should be treated as match patterns or literal values.

For example, tmp_ would mean remove all headers that begin with tmp_ including the header named tmp_. However if you want to treat * as literal value disabling this option will not perform pattern match and the only header that will be removed is the one that is an exact match.

Default is true.

Output channel

Channel where output messages should be sent after (successfully) processing the input message.

You can select the nullChannel here to silently drop the output messages.

Required

Input channel

Channel to consume the input messages from.

Required

Id

Name that uniquely identifies this flow component.

Required
