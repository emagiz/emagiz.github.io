tcp-outbound-channel-adapter

id: title: TCP outbound channel adapterTCP outbound channel adapter

An outbound channel adapter that sends TCP/IP byte-messages.

Documentation

Connection factory

A TCP connection factory is needed by an outbound adapter.

If the connection factory has a type client, the factory is 'owned' by this adapter.

If it has a type server, it is 'owned' by an inbound channel adapter and this adapter will attempt to correlate messages to the connection on which an original inbound message was received.

Client mode

If set to true, causes the adapter to establish a connection when started, rather than when the first message is sent. Requires a "client" connection factory, with single-use set to false.

Default is false.

Retry interval

When in client mode, specifies the retry interval (in milliseconds) if a connection cannot be established.

Default is 60000 (1 minute).
