SOAP action

Operation

The name of the web service operation this SOAP action applies to.

When using the default settings, the operation name is the name of your request body (as specified in the XSD) minus the Request suffix (and excluding the namespace).

SOAP action

The SOAPAction HTTP request header field can be used to indicate the intent of the SOAP HTTP request. The value is a URI identifying the intent. SOAP places no restrictions on the format or specificity of the URI or that it is resolvable.

Some examples:

  • http://www.example.org/my-service/my-operation
  • urn:uuid:6e8bc430-9c3a-11d9-9669-0800200c9a66
