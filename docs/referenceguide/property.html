Property

Is global

By default a property value applies to just a single runtime. By setting this property to yes, the property becomes available to all runtimes of the company.

Data type

Data type of the property.

  • Username: any text value, but gives the option to generate a random unique value
  • Password: any text value, but gives the options to generate a random value; also hides the value in the overview
  • String: any text value
  • Integer: a positive or negative whole number
  • Cron trigger: a text value that is a list of six fields representing a set of times

Default is string.

Name for this property. Must be a unique name within the selected runtime.

Value

Value for this property. Must be a valid integer, i.e. a whole number between -2147483648 and 2147483647.

Required

Value

Value for this property: a pattern used by a cron-trigger to specify the trigger schedule.

The pattern is a list of six single space-separated fields, representing second minute hour day month weekday. Month and weekday names can be given as the first three letters of the English names.

Example patterns:

0 0 * * * * = the top of every hour of every day 0 0 8-10 * * * = 8, 9 and 10 o'clock of every day 0 0/30 8-10 * * * = 8:00, 8:30, 9:00, 9:30 and 10 o'clock every day 0 0 9-17 * * MON-FRI = on the hour nine-to-five weekdays 0 0 0 25 12 ? = every Christmas Day at midnight
