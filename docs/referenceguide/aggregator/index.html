Aggregator

Provides different operations that allow you to summarize the values from a source list and obtain a new single value. The new value is the result of an operation applied to the elements in the list.

The Sum option returns the sum of all the source attribute values present in the source list.

The Avg option returns the average of all the source attribute values present in the source list. This means, a sum of all values divided by the total of elements in the list.

The Min option returns the minimum value encountered in the source list for the selected source attribute.

The Max option returns the maximum value encountered in the source list for the selected source attribute.

The Position returns a specific value in the source list. You can select the head or the last position in the list.

The Count option returns the number of items in the source entity list where the selected attribute is present. Returns 0 if the source list is empty

The String join option returns a string containing all the values in the source list separated by a Separator

The XPath option allows you to fully customize the aggregation operation with a custom XPath expression. Technically, the XPath you enter will be used for the <xsl:value-of> statement in the generated stylesheet. Use the view XSLT button to see a preview how your XPath will be embedded in the resulting XSLT.

String used as a separator for the concatenation of values from the source list. For example for the following ('a', 'b', 'c') values, if the separator is ; the resulting xs:string will be a;b;c. If the separator is not provided, the values will be concatenated one next to another.
